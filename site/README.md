# Site GitHub Pages — Loups-Garous

Ce dossier contient un site statique (HTML/CSS/JS) :
- génération de compositions (6–18 joueurs)
- édition de composition personnalisée
- assistant Meneur de Jeu « tour par tour »
- éditeur d'impression

## Éditeur d'impression

La page `print.html` permet d'ajuster la mise en page pour l'impression (tailles, espacement, traits de coupe) et gère le recto/verso.
Des options permettent de gérer l'impression recto/verso (qui est optionnelle), et les décalages potentiels entre les deux faces.
